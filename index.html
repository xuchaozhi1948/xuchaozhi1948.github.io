    HTML

    Html是超文本标记语言,是通过标记符号来标记要显示的网页中的各个部分，是由浏览器解释执行的语言。HTML文本是由HTML标签组成的描述性文本。

  • 

    CSS 3

    CSS即层叠样式表，在网页制作时采用层叠样式表技术，可以有效地对页面的布局、字体、颜色、背景和其它效果实现更加精确的控制。

  • 

    移动端布局

    移动端布局的话，先得明白物理像素转换逻辑像素，而且得改变视口，改变js文件，及里面的设计稿宽度，用rem布局，而且head标签中要加meta。

  • 

    flex布局

    Flex是意为”弹性布局”，用来为盒状模型提供最大的灵活性。任何一个容器都可指定为Flex布局。可以简便、完整、响应式地实现各种页面布局。

  • 

    响应式布局

    响应式布局是2010年才提出的，简而言之就是一个网站页面可以适配多个终端，而不是为了每个终端设计不同的页面，它更加灵活，省力。

  • 

    栅格化布局

    栅格化布局就是为了更快速的实现响应式页面，它预定义类，把网页分成12份，能匹配三种终端，四种屏幕大小，且移动端优先。

  • “好物”购物APP

    开发背景

    开发背景

    “网上购物”最大的特点就体现在方便，快捷，实惠。随着电子商务的发展，越来越多的人挤进这一行列，瓜分这块大蛋糕，导致网上出现很多假冒伪劣产品，让许多消费者对网购都不敢触及，还不如传统的购物方式更加安心。那么，一款可以让消费买的放心，买的安心的App购物软件显得就十分必要了。现如今的淘宝，天猫，京东，唯品会等购物App已经在市场上占有很大的用户群，与这些大的网上购物商城其争抢这块大蛋糕，根本运作不来。但是，随着“大数据”的来临，我们乘着“大数据”时代的顺风车，开发和这些App互为补充，相得益彰的一款App——“好物购物App”。

  • “好物”购物APP

    开发目的

    开发目的

    “好物购物App”的开发，就是为了让消费者买的放心，安心。 “好物购物App”是淘宝，天猫，京东，唯品会等购物App优品集合，是一款集合购物、评论、聊天的沟通型的购物App。我们依托“大数据”时代的来临，我们把这些优秀，用户群庞大的购物App上的好评商品集合到一起，通过“大数据”分析，我们把好评的商品，分享到我们的购物App当中，又或者说，我们搜索的商品的时候会优先显示好评率在90%的商品等，我们面对的不在是商家，而是已经在其他购物App上实实在在购买了此商品的消费者；我们通过他们分享出来的好评，优秀的商品，来对商品获得认可而进行购买。

  • “好物”购物APP

    功能模块

    功能模块

    主功能界面主要包括的搜索功能，通过搜索找到用户所需产品，首页中推送的是今日热评、正品保障和最多分享的一些产品，首页，主要对一类优秀好评的产品进行推送，也包括一个搜索的功能。今日热评，正品保障，最多分享会规定有几个位置。具体推送内容会由消费者分享产生的产品条件来确定，首页中包括了这款软件的主要的功能，是该软件的主界面，包括上方的一个搜索功能，在首页上放置一些今日热评、最多分享、活动广告一类，供用户浏览选择。在首页的下方是我们的四个主要按钮，首页、好评产品、他人分享、消息、我的。还有各级分页和详情页，列表页，商品页，付款页，登录注册页等。
